# Personal Introduction Webpage

## 프로젝트 설명
Woody Song의 개인 소개 및 포트폴리오를 표시하는 단일 페이지 웹 애플리케이션입니다. 개발자와 기획자로서의 경력을 소개하며, 현대적인 디자인과 부드러운 애니메이션으로 전문성을 표현합니다.

## 기술 스택
- 순수 HTML5
- 순수 CSS3
- 순수 JavaScript (ES6+)
- 프레임워크, 빌드 도구, 외부 라이브러리 없음

## 실행 방법
웹 브라우저에서 `index.html` 파일을 직접 열면 됩니다. 설치나 빌드 과정이 필요 없습니다.

## 아키텍처
- **단일 파일 애플리케이션**: 모든 HTML, CSS, JavaScript가 `index.html`에 포함됨
- **클라이언트 전용**: 서버나 백엔드 불필요
- **반응형 디자인**: 모바일과 데스크톱 환경 모두 지원
- **스크롤 가능한 카드**: 세로 스크롤이 가능한 단일 카드 레이아웃
- **섹션별 구성**: Hero, About Me, Experience, Education, Skills 섹션으로 구성
- **애니메이션**: 페이지 로드 시 순차적 페이드인 효과 및 인터랙션 애니메이션

## 디자인 특징
- 보라색 그라데이션 배경
- 글라스모피즘 효과가 적용된 반투명 카드
- Noto Sans KR 폰트 사용
- 부드러운 섹션 전환 애니메이션
- 호버 효과가 있는 경험 카드
- 커스텀 스크롤바 디자인
